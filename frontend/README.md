# AI Agent Presentation

A React TypeScript presentation about mastering AI agents with minimal tool techniques.

## Project Structure

This project consists of:
- **Frontend**: React TypeScript application (Vite)
- **Backend**: Node.js TypeScript Express server

## Frontend Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the displayed URL (usually `http://localhost:5173`)

### Frontend Features

- Responsive presentation slide format (scales with viewport)
- Dark green gradient background
- Rounded corners with shadow effects
- Smooth animations
- TypeScript support

## Backend Setup

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Run in development mode:
```bash
npm run dev
```

The backend server runs on `http://localhost:3001` by default.

### Backend Endpoints

- `GET /health` - Health check endpoint
- `GET /api/hello` - Example API endpoint
