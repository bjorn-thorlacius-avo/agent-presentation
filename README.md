# AI Agent Presentation

A React TypeScript presentation about mastering AI agents with minimal tool techniques.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to the displayed URL (usually `http://localhost:5173`)

## Project Structure

- `src/App.tsx` - Main application component
- `src/components/Slide.tsx` - Reusable slide component
- `src/App.css` - Styling for the presentation

## Features

- 1280x1024 presentation slide format
- Dark green gradient background
- Rounded corners with shadow effects
- Smooth animations
- TypeScript support
